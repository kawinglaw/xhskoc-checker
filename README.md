# 小红书 KOC 资源查重工具

一个轻量级的 H5 页面，供团队成员在拓展小红书 KOC 前进行内部资源查重。

## 功能特性

- 📊 **数据驱动**：自动读取同目录下的 `data.csv` 文件
- 🔄 **实时更新**：每次打开页面通过时间戳绕过缓存，确保获取最新数据
- 🔍 **智能查询**：支持模糊联想输入（最多显示 10 条）和精确匹配
- ✨ **高级感 UI**：深色/浅色主题切换、平滑动画、毛玻璃效果
- 🚀 **零依赖部署**：仅需 HTML + Tailwind CSS CDN + PapaParse CDN

## 快速开始

### 1. 准备数据文件

在项目根目录下创建或更新 `data.csv` 文件，格式如下：

```csv
昵称,主页链接
小红书达人小美,https://www.xiaohongshu.com/user/profile/xxx
美妆博主 Lily,https://www.xiaohongshu.com/user/profile/yyy
```

**注意：**
- 第一行必须为表头：`昵称,主页链接`
- 每行一条记录，用逗号分隔
- 如果昵称或链接中包含逗号，需用双引号包裹

### 2. 本地测试

直接双击打开 `index.html` 文件即可在浏览器中预览（部分浏览器可能需要本地服务器）。

推荐使用 VS Code Live Server 插件或 Python 简易服务器：

```bash
# Python 3
python3 -m http.server 8080

# 访问 http://localhost:8080
```

### 3. 部署到 GitHub Pages

#### 步骤一：创建 GitHub 仓库

1. 在 GitHub 上创建一个新的公开仓库（例如 `xiaohongshu-koc-checker`）
2. 将项目文件（`index.html` 和 `data.csv`）上传至仓库根目录

#### 步骤二：启用 GitHub Pages

1. 进入仓库 → **Settings** → **Pages**
2. 在 **Source** 下拉菜单中选择 `main` 分支（或 `master`）
3. 点击 **Save**

#### 步骤三：访问页面

等待 1-2 分钟后，访问以下地址：

```
https://<你的GitHub用户名>.github.io/xiaohongshu-koc-checker/
```

### 4. 更新数据

当需要添加新的 KOC 数据时：

1. 在本地更新 `data.csv` 文件
2. 将修改后的文件推送至 GitHub 仓库
3. 团队成员刷新 H5 页面即可获取最新数据

## 使用说明

### 查询流程

1. **输入昵称**：在搜索框中输入 KOC 昵称的部分字符
2. **查看联想**：系统实时显示匹配的昵称列表（最多 10 条）
3. **点击或查询**：
   - 点击联想词自动执行精确匹配查询
   - 或手动点击"查询"按钮
4. **查看结果**：
   - ✅ **已建联**：显示红色背景，昵称作为超链接可点击跳转至主页
   - 🎯 **未建联**：显示绿色背景，提示"该KOC未建联，去拓吧！"

### 主题切换

点击右上角的 🌙/☀️ 按钮可在深色模式和浅色模式之间切换。

## 技术栈

- **HTML5**：页面结构
- **Tailwind CSS (CDN)**：样式框架
- **JavaScript (原生)**：交互逻辑
- **PapaParse (CDN)**：CSV 解析库

## 注意事项

- 本工具为纯前端实现，所有数据存储在 `data.csv` 文件中
- 由于浏览器安全策略，直接打开 HTML 文件可能无法加载 CSV（CORS 限制），建议使用本地服务器或部署到 GitHub Pages
- 数据隐私：请确保 `data.csv` 中的信息仅包含团队内部共享的 KOC 资源

## 常见问题

### Q: 为什么页面显示"数据加载失败"？

A: 请检查以下几点：
1. `data.csv` 文件是否与 `index.html` 在同一目录下
2. CSV 文件格式是否正确（UTF-8 编码，表头为"昵称,主页链接"）
3. 如果使用本地文件打开，尝试使用本地服务器运行

### Q: 如何批量导入大量 KOC 数据？

A: 可以使用 Excel 编辑后另存为 CSV 格式，确保：
- 编码为 UTF-8
- 第一行为表头
- 无多余空行

### Q: 联想功能不生效？

A: 请确认：
1. 输入的字符是否在数据库中存在（支持模糊匹配）
2. CSV 数据是否成功加载（查看浏览器控制台）
